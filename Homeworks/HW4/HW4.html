Question GE1

Let {Yt} be the AR(1) plus noise time series defined by Yt = Xt +Wt, where {Wt} WN(0, W2), {Xt} is the AR(1) process Xt \(\phi\) Xt1 = Zt,
{Zt} WN(0, Z2), |\(\phi\)|<1, and E(WsZt) = 0 for all t, s.

  1. Show that {Yt} is stationary and find its autocovariance function.
    \[ \begin{aligned} \text{Given:} &\\ &\begin{cases} X_t=\phi_1X_{t-1}+Z_t\\ |\phi_1|<1\\ Z_t=WN(0,\ \sigma_Z^2)\\ Y_t=X_t+W_t\\ W_t=WN(0,\ \sigma_W^2)\\ E[W_sZ_t]=0\\ \end{cases}\\\\ X_t\ &\text{is a stationary AR(1) process}\\ &\text{because } \phi(z) = 1-\phi_1z = 0\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 1=\phi_1z\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ 1/\phi_1 = z\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \text{if } \phi_1<1,\ \text{then } 1/\phi_1 >1,\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \text{thus }z>1\\ \end{aligned} \]
    \[ \begin{aligned} Var[X_t] &= \gamma_X(0)= \frac{\sigma_Z^2}{1-\phi_1^2}\\\\ E[Y_t]&=E[X_t+W_t]\\ &= E[X_t]+E[W_t] = 0 \\\\ Var[Y_t] &= Var[X_t+W_t]\\ &= E[(\phi_1X_{t-1}+Z_t+W_t)X_{t-k}]\\ &=E[\phi_1X_{t-1}X_{t-k}]\ +\ E[Z_tX_{t-k}]\ +\ E[W_tX_{t-k}]\\\\ &for\ \ k=0: \ \ \ \ \begin{cases} E[\phi_1X_{t-1}X_{t}] \ \ \ \ \ \ \ \ \ \ = \phi_1\gamma_X(1) \\ E[Z_tX_{t}] = E[Z_t^2] \ \ \ = \sigma_Z^2\\ E[W_tX_{t}] = E[W_t^2] \ = \sigma_W^2 \end{cases}\\\\ & Var[Y_t]=\gamma_Y(0) =\phi_1^2\gamma_X(0)+\sigma_Z^2+\sigma_W^2\\\\ \end{aligned} \]

\[ \begin{aligned} \text{If}\ \ &\phi_1^2\gamma_X(0) +\sigma_Z^2 =\frac{\phi_1^2\sigma_Z^2}{1-\phi_1^2} + \frac{\sigma_Z^2(1-\phi_1^2)}{1-\phi_1^2}= \frac{\sigma_Z^2}{1-\phi_1^2}, \\\\ \text{then}\ \ &\phi_1^2\gamma_X(0) +\sigma_Z^2+\sigma_W^2 =\frac{\phi_1^2(\sigma_Z^2+\sigma_W^2)}{1-\phi_1^2} + \frac{(\sigma_Z^2+\sigma_W^2)(1-\phi_1^2)}{1-\phi_1^2}= \frac{\sigma_Z^2+\sigma_W^2}{1-\phi_1^2} \\\\ &\gamma_Y(k)=\begin{cases}\frac{\sigma_Z^2+\sigma_W^2}{1-\phi_1^2} \ \ \ \ \ \ \ \ \ \ \ \ \text{for k=0}\\\\ \frac{\phi_1^k(\sigma_Z^2+\sigma_W^2)}{1-\phi_1^2}\ \ \ \ \ \ \ \text{for k}\geq1 \end{cases}\ \end{aligned} \] (b) Show that the time series Ut := Yt \(\phi Y_{t1}\) is 1-correlated (i.e., U(h) = 0 for h > 1) and hence is MA(1) process.
\[ \begin{aligned} U_t&=Y_t - \phi Y_{t-1}\\ &= X_t+W_t-\phi (X_{t-1}+W_{t-1})\\ &= X_t-\phi X_{t-1}+W_t-\phi W_{t-1}\\ &=(1-\phi B)(X_t+W_t)\\ \text{This is a MA(1) process with }& Z_t \text{ substituted for } X_t+W_t\\ & \&\ X_t \text{ substituted for }Y_t \end{aligned} \] (c) Conclude from (b) that {Yt} is an ARMA(1,1) process and express the three parameters of this model in terms of \(\phi\), W2 and Z2. \[ U_t =(1-\phi B)(X_t+W_t)\\ Y_t - \phi Y_{t-1} = (1-\phi B)(X_t+W_t)\\ (1-\phi B)(Y_t) = (1-\phi B)(X_t+W_t)\\ \text{Thus, it is an ARMA(1,1) Process} \]
