Question 1

Understanding deterministic and stochastic trends. You are given the following statements about time series:

    1. Stochastic trends are characterized by explainable changes in direction.
    1. Deterministic trends are better suited to extrapolation than stochastic trends.
    1. Short term extrapolation of deterministic trends can be justified by claiming that underlying trends will usually change slowly in comparison with the forecast lead time.

Determine which statements are true. Explain.

        A. I only
        B. II only
        C. III only
        D. I, II, and III
        E. The answer is not given by (A), (B), (C), or (D).


  1. False. Stochastic trends are characterized by unexplainable changes in direction. They arise from the random movement of the variable over time.

  2. True. Deterministic trends are a result of the constant effect of a few causal forces an are generally stable over time.

  3. True. Deterministic trends can provide accurate forecasts for several periods ahead, as forecasting generally assumes that trends will continue and change relatively slowly.

        For all t where t \(\ge\) 1, E[Xt] = E[Xt-1] + Zt

\[ \begin{align*} E[Z_{t}] & = E[Z_{1}] = E[Z_{2}] = E[Z_{t-1}] = E[Z_{t}] = \mu_{Z} \\ E[X_{t}] & = E[Z_{1}] + E[Z_{2}] + ... E[Z_{t-1}] + E[Z_{t}] \\ & = tE[Z_{t}] \\ & = t\mu_{Z} \\ \end{align*} \] \[ \begin{aligned} & E[X_{1}] = E[Z_{1}] = \mu_{Z}\\ & E[X_{2}] = E[X_{1}] + E[Z_{2}] = 2\mu_{Z}\\ & E[X_{t-1}] = E[X_{t-2}] + E[Z_{t-1}] = (t-1)\mu_{Z}\\ & E[X_{t}] = E[X_{t-1}] + E[Z_{t}] = t\mu_{Z}\\ \\ & Therefore,\ E[X_{1}] \neq E[X_{2}] \neq E[X_{t-1}] \neq E[X_{t}] \end{aligned} \]

Since \(\mu_{Z} > 0\ \ or \ \ \mu_{Z} < 0,\ X_{t}\) either increases or decreases by increments of \(\mu_{Z}\)

Define: \(X_t= \begin{cases}Z_t, & \text { if } t \text { is even } \\ \left(Z_{t-1}^2-1\right) / \sqrt{2}, & \text { if } t \text { is odd }\end{cases}\)

\[ \begin{align*} Z_{t} \sim WN(0, 1) \end{align*} \]

\[ X_t= \begin{cases}Z_t, & \text { if } t \text { is even } \\ \left(Z_{t-1}^2-1\right) / \sqrt{2}, & \text { if } t \text { is odd }\end{cases} \]

\[ \begin{align*} \text{If t is even}:&\\ & E[X_{t}] = E[Z_{t}]\\ &\ \ \ \ \ \ \ \ \ \ = 0\\ \text{If t is odd}:&\\ & E[X_{t}] = E[(Z^2_{t-1}-1)/\sqrt{2}]\\ &\ \ \ \ \ \ \ \ \ \ = \sqrt{2}\ (E[Z^2_{t-1}]-1)\\ &\ \ \ \ \ \ \ \ \ \ = \sqrt{2}\ (Var[Z_{t-1}]+E[Z_{t-1}]^2-1)\\ &\ \ \ \ \ \ \ \ \ \ = \sqrt{2}\ (1+(0)^2-1)\\ &\ \ \ \ \ \ \ \ \ \ = 0 \end{align*} \]

\[ \begin{align*} \text{If t is even}:&\\ & Var[X_{t}] = Var[Z_{t}]\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = 1\\ \text{If t is odd}:&\\ & Var[X_{t}] = Var[(Z^2_{t-1}-1)/\sqrt{2}]\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = \frac{1}{2}\ (Var[Z^2_{t-1}])\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = \frac{1}{2}\ (E[Z_{t-1}^4]-E[Z^2_{t-1}]^2])\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = \frac{1}{2}\ (3(Var[Z_{t-1}])^2-(Var[Z_{t-1}]-E[Z_{t-1}])^2)\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = \frac{1}{2}\ (3(1)^2-(1-0^2)^2)\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = \frac{1}{2}\ (2)\\ &\ \ \ \ \ \ \ \ \ \ \ \ \ \ = \ 1\\ \end{align*} \]

\[ \begin{align*} &E[X_{t}] = \begin{cases}E[Z_t] = 0, & \text { if } t \text { is even } \\ \left(E[(Z_{t-1}^2-1\right) / \sqrt{2}] = 0, & \text { if } t \text { is odd }\end{cases}\\ & \ \ \ \ \ \ \ \ \ \ = 0\\ \\ &Var[X_{t}] = \begin{cases}Var[Z_t] = 1, & \text { if } t \text { is even } \\ \left(Var[(Z_{t-1}^2-1\right) / \sqrt{2}] = 1, & \text { if } t \text { is odd }\end{cases}\\ & \ \ \ \ \ \ \ \ \ \ \ \ \ \ = 0\\ \\ & \rightarrow\ X_{t} \sim WN(0,1) \end{align*} \]
